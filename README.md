Sebuah platform *crowdsourced* yang bertujuan untuk membantu orang-orang melakukan sesuatu.

* React.js  

### Back-end

* Entity Framework Core (O/RM)
* AutoMapper (object-object mapper)
* PostgreSQL (RDBMS)
* NSwag (API preview and documentation)

* Docker
* Ubuntu
